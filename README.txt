O jeito mais fácil de pedir delivery de comida é por meio de sites e aplicativos. Dessa forma, este repositório possui o projeto de uma página web para delivery de comida.

A página implementada é formada por 3 tipos de arquivos: HTML, CSS e JavaScript. 

A página web de delivery possui as seguintes funcionalidades:
1 - Exibe as produtos vendidos de forma categorizada
2 - Permite que o cliente escolha a combinação desejada ao dar um click sobre o produto
3 - Ao finalizar montagem do pedido, um botão "Finalizar Compra" irá redirecionar o pedido escolhido com uma mensagem no WhatsApp da empresa
